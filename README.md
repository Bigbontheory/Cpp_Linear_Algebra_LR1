# Лабораторная работа №1: Прямые методы решения СЛАУ

Проект посвящен изучению и реализации классических прямых методов решения систем линейных алгебраических уравнений на языке C++.

## Цель работы
Изучить и реализовать основные прямые методы решения СЛАУ, а также провести сравнительный анализ их вычислительной эффективности и численной устойчивости.

---

## Реализованные методы

### 1. Метод Гаусса (классический)
Стандартный алгоритм исключения неизвестных.
* **Особенность:** Требует ненулевых элементов на главной диагонали.
* **Недостаток:** Высокая чувствительность к ошибкам округления при наличии малых диагональных элементов.

### 2. Метод Гаусса с выбором ведущего элемента (Pivoting)
Улучшенная модификация, выполняющая поиск максимального по модулю элемента в текущем столбце перед каждым шагом исключения.
* **Преимущество:** Значительное повышение численной устойчивости и минимизация накопления погрешностей.

### 3. LU-разложение (метод Дулиттла)
Разложение исходной матрицы $A$ в произведение двух треугольных матриц: $A = LU$.
* **Алгоритм:** 
    1. Нахождение матриц $L$ (нижнетреугольная) и $U$ (верхнетреугольная).
    2. Решение системы $Ly = b$ (прямая подстановка).
    3. Решение системы $Ux = y$ (обратная подстановка).

---

## Описание экспериментов

### 4.1. Время решения одиночной системы
Сравнение времени работы методов на случайных матрицах размерности от $100$ до $1000$.
* **Теория:** Оба метода имеют сложность $O(n^3)$. В LU-разложении основное время затрачивается на этап декомпозиции.

### 4.2. Эффективность на множественных правых частях
Решение системы $Ax_i = b_i$ для $k$ различных векторов $b$.
* **Суть:** В то время как метод Гаусса повторяет весь цикл вычислений $k$ раз, LU-разложение выполняется единожды. Для каждого последующего вектора $b$ выполняются только операции подстановки сложности $O(n^2)$.
* **Результат:** При больших $k$ метод LU демонстрирует многократное превосходство в скорости.

### 4.3. Оценка точности на матрицах Гильберта
Использование плохо обусловленных матриц Гильберта ($H_{ij} = \frac{1}{i+j-1}$) для проверки устойчивости.
* **Метрики:** 
    * Относительная погрешность: $\frac{\|x_{exact} - x_{calc}\|}{\|x_{exact}\|}$
    * Норма невязки: $\|Ax_{calc} - b\|$

## Выводы по работе
1. **Эффективность:** Эксперимент 4.2 наглядно подтверждает, что LU-разложение является оптимальным выбором для систем с повторяющейся матрицей коэффициентов.
2. **Устойчивость:** Выбор ведущего элемента (Pivoting) в методе Гаусса позволяет получить более точный результат на плохо обусловленных матрицах (на примере $n=10$ ошибка в 1.5 раза меньше).
3. **Критическая точка:** При размерности матрицы Гильберта $n=15$ относительная погрешность превышает 100% ($3.37e+00$), что свидетельствует о полной потере точности из-за специфики обусловленности матрицы и ограничений разрядности `double`.
